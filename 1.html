HTML

Let's learn about html. First of all, 'tag'. It's very important. 열리는 태그 is < > and 닫히는 태그 is < / >
Do you want a highlight something? Use 'strong' tag or 'u' tag. u means underline. h1 tag is for title. h means heading. There are only h1 to h6.
https://advancedwebranking.com/html/

'br' tag is for line breaking. It doesn't need 닫히는 태그. Use it once, twice or three times.

You can also use 'p' tag. In this case, it needs 닫히는 태그, but it is more important. You can use it with css which we will learn later.


Use 'img' tag when you want to put some images on your website. Do it with 'attribute'.
free image website: unsplash.com

태그에는 parent와 child 관계가 있다. 대표적으로 ul/li가 있다.
ul: unordered list, ol:ordered~, li: list
